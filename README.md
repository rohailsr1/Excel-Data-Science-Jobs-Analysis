This project analyzes the **global data job market** using a dataset of **32,672 job postings**. The analysis was performed in **Microsoft Excel**, where the dataset was cleaned, analyzed, and transformed into an **interactive dashboard**.

The goal of this project is to understand:

- Demand for different **data roles**
- **Salary distribution** across job titles
- **Country-wise salary differences**
- **Job type distribution**
- Most active **job posting platforms**

The dashboard converts raw job data into **visual insights that highlight trends in the data job market**.

---

The dataset contains **32,672 job postings related to data careers** collected from multiple job platforms.

# 🧹 Data Cleaning

Before creating the dashboard, the dataset required preprocessing.

### Cleaning Steps

1. Removed rows containing missing salary data  
2. Standardized job titles across the dataset  
3. Cleaned country and location columns  
4. Structured salary columns for analysis  
5. Organized skills data for aggregation  

After cleaning, **32,672 job records were available for analysis**.

# 📈 Analysis & Insights

## 1. Job Demand by Role

The dataset shows strong demand for several key roles:

- Data Analyst  
- Data Scientist  
- Data Engineer  
- Machine Learning Engineer  
- Business Analyst  

These roles account for **a large portion of the 32,672 job postings**.

---

## 2. Salary Analysis

Salary analysis shows clear differences between roles.

Key findings:

- Machine Learning Engineers often earn **over $150,000 annually**
- Data Engineers frequently earn **above $120,000**
- Data Analysts typically earn **between $60,000 and $90,000**

This highlights the **salary premium for advanced technical roles**.

---

## 3. Salary by Country

Salary levels vary significantly across countries.

Key insights:

- The **United States offers the highest salaries** in the dataset
- Developed technology markets show **higher compensation levels**
- Salary differences are strongly influenced by **regional demand and economic conditions**

---

## 4. Job Type Distribution

Employment types in the dataset include:

- Full-time  
- Contract  
- Internship  
- Temporary  

The majority of job postings are **full-time roles**, indicating long-term demand for data professionals.

---

## 5. Job Platform Distribution

The dataset includes job postings from major job platforms:

- LinkedIn  
- Indeed  
- ZipRecruiter  
- Ai-Jobs  
- Snagajob  

**LinkedIn and Indeed host the largest share of job postings** in the dataset.

---

# 💡 Key Insights

From the analysis, several trends emerge:

1. Data Analyst, Data Scientist, and Data Engineer roles dominate job postings.  
2. Machine Learning Engineers and Data Engineers earn the highest salaries.  
3. The United States offers significantly higher salaries than most countries.  
4. Full-time roles represent the majority of job postings.  
5. LinkedIn and Indeed are the most active hiring platforms.
