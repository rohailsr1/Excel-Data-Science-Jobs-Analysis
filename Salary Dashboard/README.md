📊 Data Jobs Salary Dashboard






📌 Project Overview

This project analyzes the global data job market using a dataset of 32,672 job postings. The analysis was conducted in Microsoft Excel, where the raw dataset was cleaned, transformed, and visualized in an interactive dashboard.

The objective of this project is to explore trends in the data job market and identify insights related to:

Demand for different data roles

Salary distribution across job titles

Country-wise salary differences

Job type distribution

The most active job posting platforms

The dashboard converts raw job data into clear visual insights that highlight trends in the global data job market.

📈 Analysis & Insights
1. Job Demand by Role

The dataset highlights strong demand for several key roles:

Data Analyst

Data Scientist

Data Engineer

Machine Learning Engineer

Business Analyst

These roles account for a significant portion of the 32,672 job postings, indicating strong demand for professionals in the data field.

2. Salary Analysis

The salary analysis reveals clear differences between roles.

Key findings include:

Machine Learning Engineers often earn over $150,000 annually

Data Engineers frequently earn above $120,000

Data Analysts typically earn between $60,000 and $90,000

This demonstrates the salary premium associated with specialized technical roles.

3. Salary by Country

Salary levels vary significantly across countries.

Key insights:

The United States offers the highest salary levels in the dataset

Countries with mature technology industries offer higher compensation

Salary differences are influenced by local demand and economic conditions

4. Job Type Distribution

The dataset contains several employment types:

Full-time

Contract

Internship

Temporary

The analysis shows that full-time roles represent the majority of job postings, indicating stable demand for data professionals.

5. Job Platform Distribution

Job postings in the dataset originate from major hiring platforms, including:

LinkedIn

Indeed

ZipRecruiter

Ai-Jobs

Snagajob

The analysis shows that LinkedIn and Indeed host the largest share of data job postings.

💡 Key Insights

From the analysis, several important trends emerge:

Data Analyst, Data Scientist, and Data Engineer roles dominate job postings.

Machine Learning Engineers and Data Engineers earn the highest salaries.

The United States offers significantly higher salaries than most countries.

Full-time positions represent the majority of job postings.

LinkedIn and Indeed are the most active platforms for data job listings.
